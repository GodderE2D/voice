Optional inlineVolume

inlineVolume: boolean
  • Defined in src/audio/AudioResource.ts:29

Whether or not inline volume should be enabled. If enabled, you will be able to change the volume of the stream on-the-fly. However, this also increases the performance cost of playback. Defaults to false.

Optional inputType

inputType: StreamType
  • Defined in src/audio/AudioResource.ts:16

The type of the input stream. Defaults to StreamType.Arbitrary.

Optional metadata

metadata: T
  • Defined in src/audio/AudioResource.ts:23

Optional metadata that can be attached to the resource (e.g. track title, random ID). This is useful for identification purposes when the resource is passed around in events. See AudioResource.metadata

Optional silencePaddingFrames

silencePaddingFrames: number
  • Defined in src/audio/AudioResource.ts:35

The number of silence frames to append to the end of the resource's audio stream, to prevent interpolation glitches. Defaults to 5.
